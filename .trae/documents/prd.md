## 1. 产品概述

办公室物品管理系统——面向企业内部的物品领用、归还、补充申请与异常核对平台。解决传统纸质/Excel登记效率低、信息不同步、异常难以追溯的问题，实现物品全生命周期的数字化管理。

- 目标用户：企业行政人员、普通员工、审计人员
- 核心价值：提升物品流转效率，减少丢失与浪费，确保异常可追溯

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 系统预设 | 创建和维护物品分类、存放点、责任人；执行批量操作 |
| 普通用户 | 系统预设 | 登记领用数量、用途和归还时间；查看个人领用记录 |
| 审计员 | 系统预设 | 查看异常记录和全量数据，不可修改任何记录 |

### 2.2 功能模块

1. **物品总览页面**：物品列表、搜索过滤、批量操作、提醒面板
2. **物品管理页面（管理员）**：分类管理、存放点管理、责任人管理
3. **领用登记页面（普通用户）**：领用登记、归还登记
4. **异常核对页面（审计员）**：异常记录列表、核对状态查看

## 3. 核心流程

### 3.1 物品领用流程
用户选择物品 → 填写领用数量和用途 → 设定预计归还时间 → 提交领用记录 → 库存自动扣减

### 3.2 物品归还流程
用户在领用记录中选择待归还项 → 填写实际归还数量和时间 → 提交归还 → 库存自动增加

### 3.3 批量操作边界处理流程
用户选中多条记录 → 应用筛选条件 → 部分选中记录被隐藏 → 系统提示"存在N条已选中但被筛选隐藏的记录" → 用户选择保留选择或清空隐藏记录的选择 → 执行批量操作仅作用于当前可见且选中的记录

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深石板灰(#1e293b) + 翡翠绿(#10b981)作为强调色，搭配琥珀黄(#f59e0b)作为警告色
- **按钮风格**：圆角(8px)、微阴影、hover状态加深
- **字体**：正文使用 Noto Sans SC，标题使用 Source Han Sans / Noto Sans SC Bold
- **布局风格**：左侧固定侧边栏导航 + 右侧内容区，卡片式内容展示
- **图标**：使用 lucide-react 图标库，线条风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 物品总览 | 搜索过滤栏 | 下拉选择器、搜索输入框、筛选标签 |
| 物品总览 | 物品列表 | 表格、复选框、状态标签、操作按钮 |
| 物品总览 | 批量操作栏 | 浮动操作栏、操作按钮、选中计数 |
| 物品总览 | 提醒面板 | 侧边卡片、颜色编码、计数徽章 |
| 物品管理 | 分类/存放点/责任人 | 表格、内联编辑、删除确认弹窗 |
| 领用登记 | 领用/归还表单 | 表单输入、下拉选择、日期选择、提交按钮 |
| 异常核对 | 异常列表 | 只读表格、状态筛选、详情展开面板 |

### 4.3 响应式设计
- 桌面优先设计，侧边栏固定
- 平板端侧边栏可折叠
- 移动端采用底部导航 + 全屏内容

### 4.4 关键交互细节

- 筛选切换提示：当已选中记录存在时，切换筛选条件弹出确认对话框，提供"保留选择"和"清空选择"两个选项
- 批量操作浮动栏：当有记录被选中时，底部出现浮动操作栏，显示选中数量和可用操作
- 隐藏选中提示：筛选后若存在被隐藏的选中记录，操作栏显示警告图标和数量，点击查看详情
- 提醒面板：右上角铃铛图标显示提醒数量，点击展开提醒面板
